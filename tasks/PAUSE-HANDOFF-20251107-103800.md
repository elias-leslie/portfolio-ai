## Key Decisions

### Architecture
- **5-signal index** (not 6): VIX, Momentum (SPY vs SMA_200), RSI, Put/Call, Credit Spreads
- **Market breadth DEFERRED**: Insufficient S&P 500 ticker coverage in database
- **Equal weighting**: 20% per signal (simpler than complex weighting)
- **252-day percentile window**: 1 trading year for historical context
- **PostgreSQL JSONB**: source_map stored as JSON string (json.dumps required)

### Patterns Established
- `storage.connection()` context manager (NOT `.get_connection()`)
- Direct `conn.execute()` (NOT `conn.cursor().execute()`)
- Date type imports as `date_type` to avoid Pydantic conflicts
- Celery task imports in `celery_app.py` but schedule NOT added (CRITICAL MISS)

### Issues Discovered & Handled
1. **FMP API Invalid**: S&P 500 constituent endpoint returns 401
2. **Wikipedia Blocked**: 403 on programmatic access to S&P 500 list
3. **CBOE Put/Call Dead**: CSV data stops at 2019, need alternative source
4. **Pydantic `date` Conflict**: Fixed by importing as `date_type`
5. **Connection Wrapper API**: Uses `.execute()` directly, not `.cursor()`
6. **Variable Name Collision**: `result` reused for both SQL and compute results - fixed

- ❌ Did NOT use fake/hardcoded Put/Call data
